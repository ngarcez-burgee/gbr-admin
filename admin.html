How to add a new club

  1. 1Fill in Club Name and Country — the slug and image filename are generated automatically.
  2. 2Rename your image file to match the generated filename, then save it to burgees_data/images/.
  3. 3Tag the burgee using the tagging panel on the right.
  4. 4Click Add Club.
  5. 5Click Export & Download when done — produces clubs_full.json, clubs.json and a backup spreadsheet.
  6. 6Upload all files via FileZilla.

CSV Bulk Upload

  1. 1Prepare a CSV: Club Name, Country, Region, City, Year Founded, Website, Contact Name, Contact Email. Image File is optional.
  2. 2Drag & drop the CSV into the upload zone.
  3. 3Review the preview and the image rename list — shows exactly what to rename each image file to.
  4. 4Click Import, then tag each new club via Edit.
Important: Nothing is saved automatically — Export and upload via FileZilla to go live.
